# esp32-rc-car-espnow
ESP32 RC Car using ESPNOW protocol

materials:

esp32 38 pin, l298 motor driver, joystick module, dc motor, battery 18650 (receiver), lipo battery 2s 25c (transmitter), cable jumper, car frame, breadboard
